# 功能拓展验收清单

### 商店与个性化
- [x] 完成专注轮次后，`focusCoins` 余额会增加。
- [x] 商店中能正确显示专注币余额和主题列表。
- [x] 可以使用专注币成功购买未解锁的主题。
- [x] 购买后，主题状态变为“已拥有”。
- [x] 点击“应用”按钮后，宠物背景会立即切换。
- [x] 购买或应用主题时会播放对应的音效。

### 生产力工具
- [x] 在控制面板可以添加和移除要屏蔽的网站域名。
- [x] 启动专注后，无法访问屏蔽列表中的网站。
- [x] 专注结束或暂停后，可以重新访问被屏蔽的网站。

### 宠物互动
- [x] 宠物在快乐度高时会播放开心的空闲动画。
- [x] 宠物在快乐度低时会播放失落的空闲动画。

### AI与连接性
- [x] 在设置中配置天气API信息后，可以通过聊天向宠物询问天气。
- [x] 宠物能够返回格式正确且内容真实的天气信息。

### 社交模拟
- [x] 当拥有多个宠物角色时，访客角色会偶尔出现。
- [x] 访客出现时，会显示其图像和对话气泡。
- [x] 访客在短暂显示后会自动消失。
